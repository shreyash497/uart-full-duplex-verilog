# UART Full-Duplex Transceiver (Verilog HDL)

## 📌 Overview
This project implements a **Full-Duplex UART Transceiver** using Verilog HDL.  
It supports simultaneous data transmission (TX) and reception (RX) with reliable synchronization.

## 🎯 Objective
To design and understand UART communication by building TX and RX modules from scratch.

## 📅 Progress
- Day 1: Project initialized and studied UART basics (Frame structure, Baud rate)

## 📖 UART Basics
- Start Bit → Indicates beginning of data
- Data Bits → Typically 8-bit data
- Stop Bit → Indicates end of frame
- Baud Rate → Speed of transmission (bits per second)
